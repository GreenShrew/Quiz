조합 0의 개수
-------------
### 문제

$\begin{pmatrix}n\m\\ \end{pmatrix}$ 의 끝자리 0의 개수를 출력하는 프로그램을 작성하시오.

- - -

### 입력
* 첫째 줄에 정수 n, m(0 ≤ m ≤ n ≤ 2,000,000,000, n \neq 0)이 들어온다.

- - -

### 출력
* 첫째 줄에 $\begin{pmatrix}n\m\\ \end{pmatrix}$ 의 끝자리 0의 개수를 출력한다.

- - -

### 예제 입력 1
|25 12|
|:---|

- - -

### 예제 출력 1
|2|
|:---|

- - -

### 알고리즘 분류
* 수학
* 정수론

- - -
