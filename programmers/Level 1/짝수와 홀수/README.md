짝수와 홀수
-------------
### 문제 설명

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

- - -

### 제한 조건
* num은 int 범위의 정수입니다.
* 0은 짝수입니다.

- - -

### 입출력 예
|**num**|**return**|
|:----|:----|
|3|"Odd"|
|4|"Even"|

- - -

### MJ 메모

* 그래도 이 정도면 간단하게 풀지 않았나? 싶었는데 다른 분들이 풀어낸 방법을 보고 이마를 치게 되었다.

**num % 2 == 0 ? "Even": "Odd";**
