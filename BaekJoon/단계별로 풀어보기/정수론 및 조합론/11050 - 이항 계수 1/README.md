이항 계수 1
-------------
### 문제

자연수 N과 정수 K가 주어졌을 때 이항 계수 $\begin{pmatrix}N\K\\ \end{pmatrix}$를 구하는 프로그램을 작성하시오.

- - -

### 입력
* 첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 10, 0 ≤ K ≤ N)

- - -

### 출력
* $\begin{pmatrix}N\K\\ \end{pmatrix}$를 출력한다.

- - -

### 예제 입력 1
|5 2|
|:---|

- - -

### 예제 출력 1
|10|
|:---|

- - -

### 알고리즘 분류
* 수학
* 구현
* 조합론

- - -
