이항 계수 2
-------------
### 문제

자연수 N과 정수 K가 주어졌을 때 이항 계수n $\begin{pmatrix}N\K\\ \end{pmatrix}$ 를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.

- - -

### 입력
* 첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ K ≤ N)

- - -

### 출력
* $\begin{pmatrix}N\\K\\ \end{pmatrix}$ 를 10,007로 나눈 나머지를 출력한다.

- - -

### 예제 입력 1
|5 2|
|:---|

- - -

### 예제 출력 1
|10|
|:---|

- - -

### 알고리즘 분류
* 수학
* 다이나믹 프로그래밍
* 조합론

- - -
